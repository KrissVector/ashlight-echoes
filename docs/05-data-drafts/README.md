# Data Drafts

These JSON files are migrated design data drafts, not final runtime resources.

Included:

- `characters.json`: current character stat and initial card draft.
- `cards.json`: current card and upgrade draft.
- `bonds.json`: early pair synergy draft.
- `rewards.json`: old reward-pool prototype data; compare against `docs/02-systems/roguelike-run.md` before implementation.
- `shop.json`: old shop and potion prototype data; compare against `docs/02-systems/potions.md` and economy rules before implementation.
- `maps.json`: intentionally empty in the legacy project; kept only as a reminder that map/encounter data must be redesigned for the step-based encounter model.

Do not treat these files as canonical implementation schemas. The canonical rules live in `docs/02-systems/`.
