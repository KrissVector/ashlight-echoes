## 角色定位与玩法思路

### 玲纱 — 坦克/护盾
**核心循环**：叠护盾 → 以守为攻（护盾缩放伤害）→ 嘲讽吸引火力
- AP=0 意味着护盾和伤害的 AP 缩放需要初叶 BUFF 才能解锁
- 升级路线分化：守誓路线（嘲讽+厚盾）vs 凝聚路线（护盾增幅滚雪球）
- 不动如山·反击在后期护盾堆高后反弹伤害很可观

### 灼夜 — 物理输出
**核心循环**：战斗狂热(AS+1)赚行动 → 嗜血之刃续航 → 绝刃·炎鸦永久暴击转折 → 击杀链滚雪球
- 绝刃·炎鸦使用后暴击率 5%→55%，是战斗的转折点
- 升级路线分化：流血路线（血染+血爆引爆²）vs 收割路线（击杀奖励 AS+能量+抽牌）

### 初叶 — 辅助/治疗
**核心循环**：自然祝福给队友 BUFF → 治愈之光/生命绽放续航 → 神恩降临翻盘
- 自然祝福是全队输出的核心引擎（给灼夜 ATK+8 = +44%）
- 神恩降临·裁决是最独特的机制：AP/ATK 翻倍 + 治疗牌变形为伤害牌

### 瑠璃 — 法术输出
**核心循环**：魔力涌动(AP 增幅+抽牌) → 奥术飞弹持续输出 → 虚空裂隙(异常联动 AoE) → 星陨终结
- 奥术飞弹随机目标不可控，但 3 枚飞弹费效比高
- 虚空裂隙按敌人异常种类加伤，鼓励和影月/灼夜配合先叠 debuff

### 影月 — 刺客/毒师
**核心循环**：暗影疾步(AS+附毒) → 毒雾弹叠毒 → 多回合积累 → 致命一击(不清空层数反复使用)
- 致命一击读取异常层数计算伤害但不消耗层数，越打越强
- 致命一击·灭绝计算毒/流血/脆弱/虚弱全部×300%，跨角色异常联动的终极收割
