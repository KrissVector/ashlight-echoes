# 第 1 章事件节点

> 状态：正式内容基线
> 适用范围：第 1 章「银誓余烬 / 银誓要塞遗址」事件节点
> 机器内容表：`data/events.json`

## 设计原则

第 1 章事件节点是非战斗文字交互节点。事件不是“正确答案题”，每个选项都应提供明确收益，差异来自收益类型和当前 run 需求。

事件节点的隐藏代价是机会成本：选择事件节点不会触发普通战或精英战的常规节点奖励流程，因此不会自动获得战斗节点默认的金币、药水判定和战后常规属性成长三选一。事件选项本身给予的金钱、药水、装备、属性成长或技能升级属于事件奖励，不等同于战斗节点默认奖励。

第 1 章事件选项只使用以下收益类型：

- 获得金钱。
- 获得药水。
- 获得装备。
- 获得角色属性成长。
- 获得卡牌升级。

第 1 章事件选项当前不使用额外显性减益；代价主要由事件节点机会成本和选项收益差异承担。

## 随机规则

事件中写作“随机 X 或 Y 品质”的奖励，不先按品质抽取。实现层应从所有满足品质条件的条目中等权随机抽取。

示例：

- “随机稀有或史诗药水”：从 `pool_potions_rare_or_epic_all` 中所有药水等权随机。
- “随机史诗或诅咒装备”：从 `pool_equipment_epic_or_curse_all` 中所有装备等权随机。

事件中写作“所有角色属性成长”的奖励，作用于当前 `RunState.roster` 中的全体角色，不限当前上场角色。

## 事件列表

### 断裂誓碑

银誓骑士团的誓碑倒在训练场边，碑文被虚空裂纹切成数段。玲纱认得碑尾残留的誓句，那是旧骑士团入誓时必须念出的最后一句。

选项：

| 选项 | 结果 |
|------|------|
| 擦净碑文 | 玲纱的「坚壁」获得技能升级二选一：`iron_bastion_up1` 或 `iron_bastion_up2`。 |
| 收起银誓碎片 | 获得 1 件随机史诗装备。 |

定位：牌组防御核心强化 vs 装备强化。

### 残破军械库

半塌的军械库里还压着封存补给。铁架深处有药剂箱，也有被银誓封条锁住的装备匣。

选项：

| 选项 | 结果 |
|------|------|
| 搜索药剂箱 | 获得 2 瓶随机稀有或史诗药水。 |
| 搬出封存装备 | 获得 1 件随机史诗或诅咒装备。 |

定位：稳定战术资源 vs 高波动装备收益。

### 星辉残灯

一盏旧式星辉灯仍在暗紫雾气中微弱发光。灯芯的光不像快要熄灭，反而像是在等待有人重新引导它。

选项：

| 选项 | 结果 |
|------|------|
| 稳定灯火 | 全局 ER +1。 |
| 分取星辉 | 当前 run 队伍全员 ATK +10%、AP +10%。 |

定位：循环能力成长 vs 直接输出成长。

### 封存医帐

破旧医帐被银誓封条锁住，里面还残留着灾变前的治疗物资。部分药剂已经变色，但封存柜中央仍有几瓶完好的药液。

选项：

| 选项 | 结果 |
|------|------|
| 清点补给 | 获得 100 金钱，获得 1 瓶随机药水。 |
| 使用旧药剂 | 当前 run 队伍每名角色各获得 1 次随机稀有级属性成长。 |

定位：经济与药水补给 vs 全队随机成长。

### 沉默训练场

训练场中央还刻着银誓骑士的步法线。玲纱记得这里曾经用来训练守势转换，灼夜则更在意那些能直接用于实战的发力痕迹。

选项：

| 选项 | 结果 |
|------|------|
| 复盘守势 | 当前 run 队伍全员 DEF +5。 |
| 强化攻势 | 当前 run 队伍全员 ATK +15%。 |
| 均衡训练 | 当前 run 队伍全员 DEF +2、ATK +8%。 |

定位：防御成长 vs 攻击成长 vs 均衡成长。
