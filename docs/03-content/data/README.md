# 正式内容表

本目录保存可直接迁移到实现层的数据内容表。它们不是旧原型草案，也不应再被视为降级参考。

当前内容表：

- `characters.json`：正式角色内容表。
- `cards.json`：正式卡牌内容表。
- `chapters.json`：正式章节主地点与固定步内容表。

如果这些 JSON 与同目录或 `docs/02-systems/` 的正文规则冲突，按以下方式处理：

- 数值、id、名称、初始牌、章节主地点以本目录 JSON 为准。
- 战斗结算、奖励、商店、节点生成等规则以 `docs/02-systems/` 与 `docs/06-implementation-spec/` 为准。
- 发现冲突时应修改文档或内容表，不允许在实现中临时猜测。
