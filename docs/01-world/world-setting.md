## 一、世界概览

### 世界名称

**辉烬大陆**（Luminash）

一片曾被"星辉"（古代魔力之源）滋养的广袤大陆。两百年前，一场被称为**"灭星之夜"**的灾变摧毁了星辉的核心——天穹之心，导致大陆魔力失控、文明崩塌。如今，残余的星辉碎片散落各地，既是最后的希望，也是纷争的火种。

### 核心冲突

天穹之心的碎裂释放了封印在世界根基中的**虚空之力**——一种侵蚀一切、扭曲生命的暗能量。虚空之力不断从裂隙中渗出，将生物化为疯狂的魔物，将文明遗迹化为死地。

来自大陆各地的幸存者们因各自的理由走到一起，组成队伍，在废墟中寻找星辉碎片，对抗虚空侵蚀，试图修复世界。没有一个统领全局的"主角"——每个角色都有自己的动机和故事。

> **叙事设计原则**：无玩家角色，玩家是上帝视角的策略指挥者。叙事完全由角色之间的互动驱动。

### 叙事基调

- 表层：二次元风格的冒险与羁绊，角色互动轻松有温度
- 底层：文明废墟上的重建，失去与守护的主题
- 不刻意卖惨，但角色各自背负着沉重的过去

## 二、五大地域

### 1. 白银王域（Argentum）

- **地理**：大陆中央的广袤平原与城塞群落，曾是最强大的人类王国
- **现状**：灭星之夜后王都陷落，仅存边境的几个要塞苦苦支撑。大量区域被虚空腐蚀，化为"灰烬荒原"
- **关键词**：骑士团、荣耀崩塌、灰烬与银白、废墟城堡
- **关联角色**：**玲纱**（白银骑士团末代副团长）
- **对应章节**：序章+第1章 —— 灰烬荒原（白银王域的腐蚀区域）

### 2. 焰境边疆（Pyreland）

- **地理**：大陆西部的火山地带与战争前线，赏金猎人和佣兵的活跃区域
- **现状**：虚空侵蚀与人类势力在此拉锯，战火不断。赏金猎人工会在这里维持着脆弱的秩序
- **关键词**：赏金、火焰、战场、弱肉强食、自由与暴力并存
- **关联角色**：**灼夜**（赏金猎人，在此成名）
- **对应章节**：第5章 —— 焰境边疆（最终决战之地）

### 3. 星墟书院（Astrarium）

- **地理**：大陆东北的浮空群岛与古代魔导设施废墟
- **现状**：灭星之夜中大部分设施失控，只有少数自律型造物仍在运行。书院深处封存着灭星之夜前的知识
- **关键词**：魔导书、符文、浮空岛、失落技术、人造生命
- **关联角色**：**瑠璃**（在废弃魔导图书馆中被唤醒的人造少女）
- **对应章节**：第3章 —— 奥术遗迹（星墟书院深层）

### 4. 翠梦古林（Verdantia）

- **地理**：大陆南部的原始森林，精灵族世代守护的生命之地
- **现状**：古林中的世界树是仅存的星辉天然蓄能体之一，但其生机正在衰退。精灵族为保护世界树而封闭了森林
- **关键词**：精灵、世界树、自然魔法、花与光、生命枯萎
- **关联角色**：**初叶**（离开森林寻找拯救枯萎之树方法的年幼精灵）
- **对应章节**：第2章 —— 翠梦古林

### 5. 暗月隐里（Kageraku）

- **地理**：大陆北部的深山峡谷与地下洞窟网络
- **现状**：影族忍村在灭星之夜中被虚空侵蚀全灭，仅一人幸存。如今这片区域已成为虚空浓度最高的禁区
- **关键词**：忍者、暗杀、月光、灭族、毒与暗影
- **关联角色**：**影月**（灭族后唯一幸存的沉默忍者）
- **对应章节**：第4章 —— 暗月隐里

## 三、角色扩展背景

### 玲纱（Reisa） —— 不碎之盾

白银骑士团曾是白银王域的精锐力量，守护王都长达三百年。灭星之夜中，虚空魔物从地底涌出，骑士团在护送平民撤退的最后一战中几乎全员战死。团长莱恩哈特·银盟（Reinhardt Silversworn）——玲纱的导师，既像父亲又像兄长一样的存在——独自面对虚空裂隙，用尽全力封堵但最终被吞噬。临终前他将骑士团的徽章——一枚水晶盾章——交给副团长玲纱，嘱托她"活下去，守护能守护的一切"。如今被虚空驱动的莱恩哈特以「断誓之盾」的形态游荡在灰烬荒原，成为第1章的最终 Boss。

此后两百年，玲纱独自流浪于灰烬荒原的边缘，守护零散的幸存者聚落。她的铠甲早已布满裂痕，但盾牌从未放下。（玲纱为长寿种族，经历了灭星之夜至今的两百年。）

- **性格**：坚毅寡言，责任感极强，对队友过度保护。偶尔露出温柔的一面，会在篝火旁默默帮人修补装备。
- **内心冲突**：对自己"活下来而战友死去"怀有深重的幸存者愧疚。
- **在队伍中的角色**：序章的核心视角角色，是队伍的精神支柱和守护者。

### 灼夜（Shakuya） —— 灼热双刃

出生于焰境边疆一个被战火烧毁的小村庄，自幼在废墟中求生。13 岁时独自猎杀了一只入侵村庄的虚空狼，从此在赏金猎人工会崭露头角。她给自己的双刀取名"朝焰"和"暮烬"，象征日出的希望和日落的终结。

灼夜追求极致的速度和力量，信奉"在敌人反应过来之前结束一切"。她接下了猎杀虚空魔物的悬赏不是为了正义，而是为了证明自己——"这个世界夺走了我的一切，但我要从它手里抢回十倍。"

- **性格**：张扬自信，好胜心强，嘴硬心软。嘴上说着"只是为了赏金"，但经常免费帮助弱者。
- **内心冲突**：用强硬伪装脆弱，害怕再次失去所以不愿与人建立深厚的羁绊。
- **在队伍中的角色**：序章中途加入（因赏金任务来到灰烬荒原与玲纱结识），第5章（焰境边疆）完成个人弧光——从"只为赏金"到"为了守护"。

### 瑠璃（Ruri） —— 星蓝之页

瑠璃是灭星之夜前由星墟书院的大魔导师们创造的"知识守护者"——一种人造生命，使命是在紧急情况下保存和传承书院的核心知识。灭星之夜中书院崩塌，瑠璃进入沉睡状态，直到两百年后被一次意外的星辉碎片共振唤醒。

醒来后的瑠璃发现世界已面目全非。她的记忆中存储着大量灭星之夜前的知识，但不包含灾变的原因。悬浮在她身边的魔导书——"星典"——是一本有自我意识的古代造物，会自主翻页、提供信息，偶尔对瑠璃的行为发出吐槽。

- **性格**：冷静理性，对世界充满好奇（像初次体验一切的孩子），说话简洁但偶有冷幽默。
- **内心冲突**：作为人造生命，不确定自己是否拥有"真正的感情"，对"活着的意义"有持续的思考。
- **在队伍中的角色**：第3章（奥术遗迹）加入。记忆中存储着灭星之夜前的知识，是揭开历史真相的关键。

### 初叶（Hatsuba） —— 初绽之芽

翠梦古林的精灵族与世隔绝，依靠世界树的星辉维持生息。但近年来世界树的生机急剧衰退，枝叶枯黄、花朵凋零，精灵长老们束手无策。年幼的初叶在一次与世界树的"对话"中感知到——世界树的根系深处被虚空之力侵蚀，唯一的拯救方法是找到散落在大陆各地的星辉碎片来净化根系。

尽管长老们反对，初叶还是偷偷离开了古林，踏上了旅途。她是第一个主动离开翠梦古林的精灵族人，身上带着世界树最后一朵花编成的花冠。

- **性格**：天真活泼但不莽撞，有超越年龄的坚定。喜欢和植物说话，会为每一朵花的凋零难过。
- **内心冲突**：离开家乡的愧疚，与"只有离开才能拯救"的矛盾。担心自己不够强。
- **在队伍中的角色**：第2章（翠梦古林）加入。队伍中最年幼但最坚定的成员，是团队温暖的来源。

### 影月（Kagegetsu） —— 残月之影

影族是暗月隐里深处的忍者一族，世代守护着一处虚空裂隙的封印。灭星之夜中，裂隙失控扩大，虚空之力吞噬了整个忍村。影月是当晚执行外出任务的唯一族人，回来时只看到被虚空完全吞噬的空洞——连废墟都没有留下。

此后影月成为大陆上最沉默的女猎手，独自追踪一切与虚空相关的线索。她相信灭族的真相并非简单的灾变，而是有人蓄意破坏了封印。（影族为长寿的暗影种族，影月亲历了灭星之夜。）

- **性格**：极度沉默，只在必要时说话，说话必有分量。行事决绝冷酷，但对无辜者绝不出手。
- **内心冲突**：复仇是唯一的动力，但旅途中逐渐感受到"活着不仅是为了报仇"。
- **在队伍中的角色**：第4章（暗月隐里）加入。追查灭族真相是她的核心动机，与队伍的主线探索目标一致。

## 四、关键名词

| 名词 | 说明 |
|------|------|
| 星辉（Lumina） | 大陆的魔力之源，来自天穹之心的能量。散落为碎片后成为各方争夺的资源 |
| 天穹之心（Celestial Core） | 曾悬浮在大陆中央上空的巨型星辉结晶，灭星之夜中碎裂 |
| 灭星之夜（The Starfall） | 两百年前的世界级灾变，天穹之心碎裂，虚空之力释放 |
| 虚空之力（Void Force） | 侵蚀一切的暗能量，将生物扭曲为魔物，将大地化为死地 |
| 星辉碎片（Lumina Shard） | 天穹之心的碎片，肉鸽中的核心收集品/货币 |
| 星辉碎片共鸣 | 碎片之间会产生共鸣指引方向，是队伍旅程的导航依据 |
| 虚空裂隙（Void Rift） | 虚空之力涌入现实世界的通道，大小不一 |
| 虚空魔物（Void Beasts） | 被虚空之力扭曲的生物，或从裂隙中涌出的怪物 |

## 五、虚空魔物体系

虚空魔物是游戏中所有敌人的统称，分为三类：

### 1. 灰烬种（Ashen Type）

被虚空之力腐蚀的原住民残骸或灵魂，保留了生前的部分形态但已被扭曲。常见于曾有文明的废墟区域。

- 视觉特征：灰白色调为主，身上有紫黑色虚空纹路，半透明/残破感
- 代表：灰烬游魂（普通小怪）、堕落骑士（精英怪）

### 2. 裂隙种（Rift Type）

直接从虚空裂隙中诞生的纯粹魔物，没有生前形态，造型怪异扭曲。

- 视觉特征：深紫/暗红色调，几何形体与有机体混合，发光的虚空核心
- 代表：更高级的敌人（后续地图）

### 3. 堕落种（Fallen Type）

曾经强大的存在（英雄、守护者、古代造物）被虚空之力彻底侵蚀后的形态，保留了原本的力量但意识被扭曲。通常作为 Boss 出现。

- 视觉特征：保留原始形态的轮廓但被黑紫色虚空晶体覆盖，眼部发出邪光
- 代表：各章 Boss（待定）

## 六、五章叙事框架

### 主线叙事结构：渐进式认知

队伍对灭星之夜真相的理解随章节推进而变化，但本版游戏不给出最终定论（为DLC/后续更新留空间）：

| 阶段 | 认知变化 |
|------|---------|
| 序章+第1章 | "这是一场天灾的余波，我们必须生存下去" |
| 第2章 | "虚空的扩散似乎有规律……它在朝星辉集中地蔓延" |
| 第3章 | "书院的记录显示，虚空对守护者的打击不像是随机的" |
| 第4章 | "封印是从虚空侧被打开的——虚空有意识？" |
| 第5章 | "碎片在引导我们？还是在考验我们？……没有答案，但我们选择继续前行" |

> 每一阶段的"真相"都是角色们的推测，不是叙事者的陈述。后续内容可以推翻、深化或引向任何方向。

### 叙事呈现方式：中等叙事

- 章节开头/结尾有短过场对话
- Boss 前后有剧情对话
- 角色入队时有入队对话
- 角色之间有互动和碰撞

### 叙事焦点轮转

| 章节 | 叙事焦点角色 | 驱动力 |
|------|------------|--------|
| 序章+第1章 | 玲纱（+灼夜） | 守护家园 / 赏金任务 → 发现更大的威胁 |
| 第2章 | 初叶 | 拯救世界树 |
| 第3章 | 瑠璃 | 寻找灭星之夜的真相 |
| 第4章 | 影月 | 追查灭族仇人 |
| 第5章 | 灼夜（全员） | 守护 / 终结虚空之源 |

### 序章+第1章：灰烬荒原（白银王域）

- **背景**：白银王域被虚空腐蚀后的废墟区域
- **叙事线**：玲纱在废墟中守护幸存者，灼夜因赏金任务来到此地，二人在对抗虚空威胁的过程中结成同盟
- **Boss**：待定（各章 Boss 是各区域被虚空侵蚀的最强存在，也可以是"击败→净化→入队"的未来队友）
- **主色调**：灰白、银色、暗紫

### 第2章：翠梦古林

- **背景**：精灵族守护的原始森林，世界树正在枯萎
- **叙事线**：队伍到达翠梦古林，发现世界树根系被虚空定向侵蚀。初叶加入队伍
- **叙事发现**：虚空的扩散似乎有规律，在朝星辉集中地蔓延
- **Boss**：待定
- **主色调**：翠绿、嫩黄、枯褐（健康与腐蚀的对比）

### 第3章：奥术遗迹（星墟书院）

- **背景**：星墟书院深层，充满失控的魔导装置和自律型防卫造物
- **叙事线**：瑠璃被碎片共振唤醒，记忆中存储着灾前知识。队伍在书院记录中发现异常
- **叙事发现**：虚空对守护者的打击不像随机的，似乎有某种选择性
- **Boss**：待定
- **主色调**：深蓝、紫、星光金

### 第4章：暗月隐里

- **背景**：影族忍村的废墟，虚空浓度最高的禁区
- **叙事线**：影月追查灭族真相，发现封印是从虚空侧被打开的
- **叙事发现**：虚空可能有意识？封印被定向破坏的证据
- **Boss**：待定
- **主色调**：深紫、暗银、月光白

### 第5章：焰境边疆

- **背景**：灼夜的故乡，人类与虚空的最前线
- **叙事线**：灼夜面对自己逃离故乡的过去，全员集结最终决战
- **叙事发现**：碎片似乎在引导队伍？在考验他们？没有确切答案
- **Boss**：待定
- **主色调**：赤红、橙黄、虚空黑

### 角色代际结构

队伍中有灭星之夜的亲历者和灾后出生的新一代，形成代际视角碰撞：

- **亲历者（200年前）**：玲纱（长寿种族）、影月（暗影种族）、瑠璃（人造生命，沉睡200年）
- **灾后新生代**：灼夜（人类，20多岁）、初叶（年幼精灵，近几十年出生）

## 七、世界观与视觉风格呼应

| 世界元素 | 视觉表达 |
|---------|---------|
| 星辉（正面能量） | 明亮的金/白/青光芒，晶体质感，温暖 |
| 虚空之力（负面能量） | 深紫/暗红的扭曲纹路，裂痕/侵蚀质感，不安 |
| 白银王域 | 银白色金属质感，中世纪骑士美学，残破但庄严 |
| 焰境边疆 | 赤红/橙黄色火焰，粗犷的战场风格 |
| 星墟书院 | 星空蓝/魔法紫，浮空的符文和书页，神秘学术感 |
| 翠梦古林 | 翠绿/嫩黄色自然色调，精灵族的精致与自然融合 |
| 暗月隐里 | 深紫/暗银/月光白，忍者的阴影美学 |
