# 装备系统设计

> 版本：v2.0
> 状态：框架和完整装备列表已确定
> 更新日期：2026-03-05

## 概述

装备是肉鸽 run 中除卡牌升级和属性成长外的第三条成长线，也是**每局卡组产生变化的核心手段**。

## 基础规则

- **3 个通用槽位**：每个角色可装备 3 件，不分类型，自由搭配
- **装备锁定**：装备一旦装上**不可自由拆卸**，只能在获得新装备时选择替换
- **商店回收**：商店中可以 **40%** 售价回收装备。诅咒装备需**花费 25 金币**才能处理（商店收取处理费）
- **所有装备都有涌现效果**：史诗/传说/诅咒装备必须附带卡牌（新增或改变现有卡牌）
- **无专属装备**：所有装备任何角色都能使用。角色流派分化由卡牌升级系统负责

## 稀有度分层（5 档）

| 稀有度 | 效果类型 | 卡牌影响 | 设计定位 |
|--------|---------|---------|---------|
| **普通** | 固定数值加成 | 无 | 前期填充 |
| **稀有** | 较强的固定或百分比加成 | 无 | 中期主力 |
| **史诗** | 属性 + 特殊技能（部分带轻度负面） | 附带/改变卡牌 | 局部改变战斗玩法 |
| **传说** | 强属性 + 全局机制改变 | 附带强力卡牌 | 改变整局构筑 |
| **诅咒** | 强属性 + 全局机制改变 + 致命负面 | 添加诅咒牌 | 传说级强度，严重 debuff |

### 装备与卡牌的联动方式

| 方式 | 说明 |
|------|------|
| **附带卡牌** | 装备后向该角色牌池加入 1-2 张新卡。替换装备则移除 |
| **改变卡牌** | 装备后改变该角色现有卡牌的效果（增强/附加效果） |
| **添加诅咒牌** | 诅咒装备向牌库加入负面卡牌，占据手牌位 |

## 装备管理规则

1. **获得时决定**：获得新装备时立刻选择装给谁、替换哪个槽位（或放弃）
2. **装备锁定**：装上后不可自由拆卸，只能被新装备替换
3. **商店回收**：商店中可以 40% 售价回收已装备的物品（替换为空槽）。诅咒装备需花费 25 金币处理
4. **卡牌跟随**：装备附带的卡牌在装备被替换/出售后从卡组中移除
