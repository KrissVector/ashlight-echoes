# 卡牌系统设计

## 概述

卡牌是战斗中的核心操作手段，角色通过打出卡牌来释放各种效果。

## 卡牌构成（已确定）

每个角色 **4 张固有初始卡牌**（基础牌池无副本）：
- 3 张非大招牌（2普通 + 1稀有）
- 1 张大招牌（史诗，初始锁定）

> **大招牌初始锁定**，需通过本章剧情战斗解锁。
> 每局 run 都需重新解锁，但**局外成长可让大招初始就解锁**。

| 角色 | 非大招（各×1） | 大招（×1，需解锁） |
|------|--------------|------------------|
| 玲纱 | 盾击、坚壁、以守为攻 | 不动如山 |
| 灼夜 | 战斗狂热、嗜血之刃、血刃旋风 | 绝刃·炎鸦 |
| 初叶 | 治愈之光、自然祝福、生命绽放 | 神恩降临 |
| 瑠璃 | 奥术飞弹、魔力涌动、虚空裂隙 | 星陨 |
| 影月 | 暗影疾步、毒雾弹、幻影步 | 致命一击 |

### 上场限制与卡组规模

- **最多 3 人上场**（序章教程除外：教程战1固定1人，教程战2固定2人）
- 解锁超过 3 个角色后，每场战斗前选择带谁
- 战斗难度按 3 人编制平衡

```
基础卡组（3人无大招）= 9 张
基础卡组（3人有大招）= 12 张
装备/药水添加卡牌后 = 约 14-17 张（随 run 决策变化）
```

卡组变化来源：
- ①选谁上场 → 决定基础 9-12 张牌
- ②卡牌升级方向 → 决定每张牌的具体效果
- ③装备附带/改变卡牌 → 装备期间改变该角色牌池
- ④药水添加/复制/删除/改造卡牌 → 永久改变该角色牌池

> **固有卡牌池基础固定**：角色自身不会自然新增固有卡牌（除剧情解锁大招）。
> 卡组变化来自装备、药水和卡牌升级等外部系统。商店不直接出售卡牌，也不提供直接删牌服务；但商店可概率出售传说药水，从而间接实现卡牌复制、删除或改造。具体规则见药水系统。

## 卡牌来源

| 来源 | 说明 |
|------|------|
| 角色固有初始卡牌 | 每个角色自带 4 张（含锁定的大招） |
| 精英战触发 | 卡牌升级（二选一方向），不是获得新卡 |
| 剧情战斗 | 解锁大招牌 |
| 装备附带 | 史诗/传说/诅咒装备向卡组添加新卡牌或改变现有卡牌 |
| 药水改造 | 稀有以上药水可添加、复制、删除或改造卡牌 |
| 商店购买药水 | 商店不直接出售或删除卡牌，但可概率出售传说药水，间接触发卡牌复制、删除或改造 |

## 卡牌与角色的关系

- 卡牌**归属于角色**，不同角色有不同的牌池
- 装备附带的卡牌归属于装备角色
- 药水新增的卡牌归属于使用时指定的角色
- 战斗中所有出战角色的卡牌**混合成公共牌库**（已确定）
- 采用四区模型：牌库（Deck）→ 手牌（Hand）→ 弃牌堆（Discard）→ 消耗堆（Exhaust）
- Deck 空时，Discard 洗牌回 Deck；Exhaust 中的卡牌本局不再回归

## 卡牌属性

| 属性 | 说明 |
|------|------|
| 名称 | 卡牌名称 |
| 归属角色 | 该卡牌所属的角色 |
| 能量消耗 | 使用该卡牌消耗的公共能量点数（2~7） |
| 伤害类型 | 物理（基于 ATK）/ 法术（基于 AP）/ 固定 / 无 |
| 效果 | 卡牌的具体效果描述 |
| 稀有度 | 普通 / 稀有 / 史诗 |
| 目标 | 单体敌人 / 全体敌人 / 单体友方 / 全体我方 / 自身 |
| 消耗性 | 是否为消耗类（使用后进入 Exhaust 而非 Discard） |

## 卡牌伤害计算（v3）

卡牌伤害由卡牌描述中的公式决定，常见模式：

- **物理伤害**：`round(atkScaling × ATK + apScaling × AP)` — 减去目标 DEF
- **法术伤害**：`round(base + apScaling × AP)` — 减去目标 DEF
- **固定/毒/流血伤害**：无视 DEF
- **治疗**：`round(base + apScaling × AP)` — 不受 DEF 影响
- **护盾**：`round(base + apScaling × AP)`
- 所有数值四舍五入取整

部分卡牌还可基于**护盾值**（shieldScaling）或**流血层数**（bleedScaling）计算伤害。

## 卡牌升级系统（已确定）

### 基本规则

- 每张卡牌有 **2 个升级方向**，主要通过精英战触发（与精英类型关联）
- 每张卡牌**只能升级 1 次**，二选一
- 升级后卡牌名称变化（如"盾击"→"盾击·强袭"或"盾击·铁壁"）
- 升级可能改变：效果数值、增加新效果、改变费用、改变消耗性

### 升级方向设计哲学

每张卡牌的两个升级方向通常代表不同的策略选择：

| 升级方向 | 常见模式 | 举例 |
|----------|---------|------|
| 方向 A | 强化核心效果 / 追加攻击性 | 盾击·强袭（提升伤害倍率） |
| 方向 B | 增加防御/联动 / 追加功能性 | 盾击·铁壁（护盾缩放伤害） |

### 特殊升级类型

- **消耗型升级**：部分卡牌升级后变为消耗型（如神恩降临的两个升级），效果更强但一局只能用一次
- **卡牌变形**：极少数升级会将其他卡牌永久变形为新卡牌（如神恩降临·裁决）
