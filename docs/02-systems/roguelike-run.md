# 肉鸽系统设计

> 版本：v3.0
> 状态：核心框架和奖励体系已确定
> 更新日期：2026-03-16

## 概述

肉鸽（Roguelike）要素为游戏提供每局不同的体验，通过随机性创造策略深度和重玩价值。

## 核心设计哲学（已确定）

> **不设硬上限，用稀有度控制平衡。**
>
> 越强的成长选项越稀有，但玩家一旦遇到就会获得极强的正反馈和爽感。
> 我们在做游戏，平衡性要考虑，但玩家的正反馈同样重要。
> "神级 run" 是玩家截图分享的素材，是肉鸽的核心魅力。

## HP 策略（已确定）

> **每场战斗开始时 HP 回满。**
>
> 面向多平台（含手游）用户，避免长线 HP 管理的挫败感。
> 策略深度不来自 HP 管理，而来自**环境修正系统**（buff/debuff）。
> 完整核心流程见 [Run 流程规格](../06-implementation-spec/run-flow.md#核心玩法循环)。

## 奖励体系（已确定）

### 属性成长机制

所有战斗胜利后均提供**三选一属性成长**：

- 三个选项各自随机生成：**随机角色** × **随机属性** × **随机数值**（按稀有度决定）
- 三个选项**独立可刷新**：每个选项可单独花费金币刷新 1 次（最多刷新 3 次/次奖励）
- 所有成长属性**无硬上限**，玩家可在一局中反复提升同一属性

### 普通战斗奖励

| 奖励 | 内容 |
|------|------|
| 属性成长 | 三选一，稀有度：**普通 80% / 稀有 20%** |
| 药水 | **30% 概率**掉落 1 瓶（普通 70% / 稀有 30%） |
| 金币 | 基础战斗金币 |

### 精英战斗奖励

| 奖励 | 内容 |
|------|------|
| 属性成长 | 三选一，稀有度：**稀有 80% / 史诗 20%** |
| 装备 | 必定掉落，品质按章节（见 [equipment.md](equipment.md)） |
| 卡牌升级 | 部分精英可触发对应角色的卡牌升级（二选一，每牌限 1 次） |
| 药水 | **30% 概率**掉落 1 瓶（稀有 70% / 史诗 30%） |
| 金币 | 基础战斗金币（比普通战多） |

### Boss 战奖励

| 奖励 | 内容 |
|------|------|
| 属性成长 | 三选一，稀有度：**史诗 75% / 传说 25%** |
| 装备 | 必定掉落，品质按章节（见 [equipment.md](equipment.md)） |
| 药水 | **必定掉落** 1 瓶（史诗 75% / 传说 25%），背包满时当场使用或丢弃 |
| 金币 | 大量金币 |

### 入队战奖励

| 奖励 | 内容 |
|------|------|
| 战前成长 | **固定为入队角色**的属性成长 N 次（三选一，可单独刷新，稀有度：**普通 50% / 稀有 30% / 史诗 20%**） |
| 角色入队 | 新角色正式加入队伍 |
| 其他 | 按精英战标准发放装备和金币 |

追赶成长次数：初叶 3 次 / 瑠璃 6 次 / 影月 9 次。详见 [map-encounters.md](map-encounters.md) 的入队战设计。

## 属性成长池（已确定）

### 普通成长（9 个选项）

稳定小成长，单次提升约 10-20% 基础值。

| # | 成长项 | 数值 | 归属 |
|---|--------|------|------|
| 1 | ATK | +3 | 单个角色 |
| 2 | AP | +3 | 单个角色 |
| 3 | DEF | +2 | 单个角色 |
| 4 | HP | +15 | 单个角色 |
| 5 | CRIT | +3% | 单个角色 |
| 6 | CDMG | +15% | 单个角色 |
| 7 | EVD | +3% | 单个角色 |
| 8 | ACC | +5% | 单个角色 |
| 9 | 金币 | +20 | 即时获得 |

### 稀有成长（9 个选项）

有感的中幅提升，出现时值得认真考虑。

| # | 成长项 | 数值 | 归属 |
|---|--------|------|------|
| 1 | ATK | +6 | 单个角色 |
| 2 | AP | +6 | 单个角色 |
| 3 | DEF | +5 | 单个角色 |
| 4 | HP | +30 | 单个角色 |
| 5 | CRIT | +8% | 单个角色 |
| 6 | CDMG | +35% | 单个角色 |
| 7 | EVD | +8% | 单个角色 |
| 8 | SPD | +2 | 单个角色 |
| 9 | 金币 | +50 | 即时获得 |

### 史诗成长（9 个选项）

改变打法级别的强力提升。

| # | 成长项 | 数值 | 归属 |
|---|--------|------|------|
| 1 | ATK | +12 | 单个角色 |
| 2 | AP | +12 | 单个角色 |
| 3 | DEF | +8 | 单个角色 |
| 4 | HP | +60 | 单个角色 |
| 5 | AS | +1 | 单个角色 |
| 6 | CRIT | +15% | 单个角色 |
| 7 | CDMG | +60% | 单个角色 |
| 8 | ER | +1 | 全局 |
| 9 | 全属性小幅 | ATK/AP +3, DEF +2, HP +15 | 单个角色 |

### 传说成长（10 个选项）

"这局稳了"的逆天体验。以**百分比成长**为主（除 DEF/AS/ER/能量上限），越到后期价值越高。

| # | 成长项 | 数值 | 归属 |
|---|--------|------|------|
| 1 | AS | +2 | 单个角色 |
| 2 | ER | +2 | 全局 |
| 3 | 能量上限 | +1 | 全局 |
| 4 | ATK | +15% | 单个角色 |
| 5 | AP | +15% | 单个角色 |
| 6 | DEF | +15 | 单个角色（减法模型保持固定值） |
| 7 | HP | +25% | 单个角色 |
| 8 | CRIT | +30% | 单个角色 |
| 9 | CDMG | +80% | 单个角色 |
| 10 | 全属性大幅 | ATK+8%, AP+8%, HP+10%, AS+1, DEF+5, CRIT+15%, CDMG+30% | 单个角色 |

### 关键成长属性的稀有度定位

| 成长项 | 稀有度 | 说明 |
|--------|--------|------|
| AS +1 | 史诗 | 行动次数极其重要 |
| AS +2 | 传说 | 同上 |
| ER +1 | 史诗 | 全局效果，影响所有角色 |
| ER +2 | 传说 | 同上 |
| 能量上限 +1 | 传说 | 全局效果，极强 |
| 全属性大幅 | 传说 | 全池最强单选项 |

> 所有成长属性无硬上限，玩家可以在一局 run 中反复提升同一属性。
> 但由于高稀有度选项出现概率低，实际上形成了"软上限"。

## 金币经济体系（已确定）

### 金币收入

| 来源 | 金额 |
|------|------|
| 普通战胜利 | +10 |
| 精英战胜利 | +30 |
| Boss 战胜利 | +100 |
| 属性成长选金币（普通池） | +20 |
| 属性成长选金币（稀有池） | +50 |
| 事件节点 | 由 `docs/03-content/data/events.json` 的具体事件选项定义 |

### 金币支出

| 用途 | 金额 |
|------|------|
| 刷新属性成长 | 5 / 次（每选项最多刷 1 次） |
| 商店相关支出 | 见 [商店系统](shop.md) |
| 事件节点 | 由 `docs/03-content/data/events.json` 的具体事件选项定义 |

### 经济节奏

- 单章基础收入约 **230-250** 金币（5 普通+2 精英+Boss+少量金币成长选项）
- 普通战刷新 3 次成本（15）> 普通战收入（10）→ 迫使玩家在刷新和攒钱之间取舍
- 商店价格、回收和品质节奏见 [商店系统](shop.md)

## 装备系统

详见 [equipment.md](equipment.md)。

核心规则：
- **3 个通用槽位**，无专属装备，任何角色可用
- **装备锁定**：装上后不可自由拆卸，只能被新装备替换
- 商店相关回收和处理规则见 [商店系统](shop.md)
- **5 档稀有度**：普通 / 稀有 / 史诗 / 传说 / 诅咒
- **史诗以上必有卡牌影响**：附带/改变/添加诅咒牌

## 药水系统

详见 [potions.md](potions.md)。

核心规则：
- **背包上限 3 瓶**（全队共享）
- **永久成长药水在节点之间使用**
- **一次性战斗药水在战斗中我方行动阶段使用，不消耗 AS**
- **永久成长效果不可撤销**
- **4 档稀有度**：普通 / 稀有 / 史诗 / 传说

## 商店系统

商店商品、服务、价格、品质分布和禁止项见 [商店系统](shop.md)。

## 局外成长（Meta Progression）

游戏失败后重新开始，但以下进度会保留：

| 成长项 | 说明 |
|--------|------|
| 角色解锁 | 通关对应章节后永久解锁新的角色可供选择 |
| 大招自动解锁 | 局外解锁角色后，对应大招开局即可用 |
| 初始属性增加 | 累计游玩可逐步提升角色的初始属性 |
| 初始 ER 提升 | 累计游玩可提升初始每回合能量恢复 |

> 更多局外成长要素待设计，如：
> - 解锁新的卡牌池
> - 成就系统
> - 挑战模式
> - 具体的货币、升级路径和解锁条件
