# 肉鸽系统设计

> 版本：v3.0
> 状态：核心框架和奖励体系已确定
> 更新日期：2026-03-16

## 概述

肉鸽（Roguelike）要素为游戏提供每局不同的体验，通过随机性创造策略深度和重玩价值。

## 核心设计哲学（已确定）

> **不设硬上限，用稀有度控制平衡。**
>
> 越强的成长选项越稀有，但玩家一旦遇到就会获得极强的正反馈和爽感。
> 我们在做游戏，平衡性要考虑，但玩家的正反馈同样重要。
> "神级 run" 是玩家截图分享的素材，是肉鸽的核心魅力。

## HP 策略（已确定）

> **每场战斗开始时 HP 回满。**
>
> 面向多平台（含手游）用户，避免长线 HP 管理的挫败感。
> 策略深度不来自 HP 管理，而来自**环境修正系统**（buff/debuff）。
> 详见 [map-design.md](map-design.md) 的节点推进流程。

## 奖励体系（已确定）

### 属性成长机制

所有战斗胜利后均提供**三选一属性成长**：

- 三个选项各自随机生成：**随机角色** × **随机属性** × **随机数值**（按稀有度决定）
- 三个选项**独立可刷新**：每个选项可单独花费金币刷新 1 次（最多刷新 3 次/次奖励）
- 所有成长属性**无硬上限**，玩家可在一局中反复提升同一属性

### 普通战斗奖励

| 奖励 | 内容 |
|------|------|
| 属性成长 | 三选一，稀有度：**普通 80% / 稀有 20%** |
| 药水 | **30% 概率**掉落 1 瓶（普通 70% / 稀有 30%） |
| 金币 | 基础战斗金币 |

### 精英战斗奖励

| 奖励 | 内容 |
|------|------|
| 属性成长 | 三选一，稀有度：**稀有 80% / 史诗 20%** |
| 装备 | 必定掉落，品质按章节（见 [equipment-system.md](equipment-system.md)） |
| 卡牌升级 | 部分精英可触发对应角色的卡牌升级（二选一，每牌限 1 次） |
| 药水 | **30% 概率**掉落 1 瓶（稀有 70% / 史诗 30%） |
| 金币 | 基础战斗金币（比普通战多） |

### Boss 战奖励

| 奖励 | 内容 |
|------|------|
| 属性成长 | 三选一，稀有度：**史诗 75% / 传说 25%** |
| 装备 | 必定掉落，品质按章节（见 [equipment-system.md](equipment-system.md)） |
| 药水 | **必定掉落** 1 瓶（史诗 75% / 传说 25%），背包满时当场使用或丢弃 |
| 金币 | 大量金币 |

### 入队战奖励

| 奖励 | 内容 |
|------|------|
| 战前成长 | **固定为入队角色**的属性成长 N 次（三选一，可单独刷新，稀有度：**普通 50% / 稀有 30% / 史诗 20%**） |
| 角色入队 | 新角色正式加入队伍 |
| 其他 | 按精英战标准发放装备和金币 |

追赶成长次数：初叶 3 次 / 瑠璃 6 次 / 影月 9 次。详见 [map-design.md](map-design.md) 的入队战设计。

### 关键成长属性的稀有度定位

| 成长项 | 稀有度 | 说明 |
|--------|--------|------|
| AS +1 | 史诗 | 行动次数极其重要 |
| AS +2 | 传说 | 同上 |
| ER +1 | 史诗 | 全局效果，影响所有角色 |
| ER +2 | 传说 | 同上 |
| 能量上限 +1 | 传说 | 全局效果，极强 |
| 全属性大幅 | 传说 | 全池最强单选项 |

> 所有成长属性无硬上限，玩家可以在一局 run 中反复提升同一属性。
> 但由于高稀有度选项出现概率低，实际上形成了"软上限"。

## 金币经济体系（已确定）

### 金币收入

| 来源 | 金额 |
|------|------|
| 普通战胜利 | +10 |
| 精英战胜利 | +30 |
| Boss 战胜利 | +100 |
| 属性成长选金币（普通池） | +20 |
| 属性成长选金币（稀有池） | +50 |
| 事件节点 | 待定（设计事件时按以上标准参考） |

### 金币支出

| 用途 | 金额 |
|------|------|
| 刷新属性成长 | 5 / 次（每选项最多刷 1 次） |
| 商店手动存档 | 10 / 次 |
| 商店净化 debuff | 25 / 次 |
| 商品购买 | 见下方价格表 |
| 事件节点 | 待定（设计事件时按以上标准参考） |

### 商店价格表

| 品质 | 装备/药水售价 | 回收价（售价的 40%） |
|------|------------|-------------------|
| 普通 | 25-50 | 10-20 |
| 稀有 | 50-80 | 20-32 |
| 史诗 | 100-180 | 40-72 |
| 传说 | 200-350 | 80-140 |
| 诅咒 | — | **-25**（花费 25 金币处理） |

### 经济节奏

- 单章基础收入约 **230-250** 金币（5 普通+2 精英+Boss+少量金币成长选项）
- 普通战刷新 3 次成本（15）> 普通战收入（10）→ 迫使玩家在刷新和攒钱之间取舍
- 传说装备（200-350）需攒约 2 章积蓄 → 与传说只出现在后期章商店（第5章5%）的设计吻合
- 装备/药水回收 40% → 补偿装备锁定不可拆卸的高犯错代价

## 装备系统

详见 [equipment-system.md](equipment-system.md)。

核心规则：
- **3 个通用槽位**，无专属装备，任何角色可用
- **装备锁定**：装上后不可自由拆卸，只能被新装备替换
- **商店回收**：40% 售价（诅咒装备花费 25 金币处理）
- **5 档稀有度**：普通 / 稀有 / 史诗 / 传说 / 诅咒
- **史诗以上必有卡牌影响**：附带/改变/添加诅咒牌

## 药水系统

详见 [potion-system.md](potion-system.md)。

核心规则：
- **背包上限 3 瓶**（全队共享）
- **只能在节点之间使用**
- **效果永久不可撤销**
- **4 档稀有度**：普通 / 稀有 / 史诗 / 传说

## 商店系统

- 通过节点描述暗示位置，不是固定间隔
- 出售内容：
  - **药水**：永久成长效果
  - **通用装备**：稳定但不如精英掉落强
  - **情报**：敌人/Boss 情报，附赠 buff 或专门道具
  - **Boss 专用道具**：影响 Boss 战的 buff/机制
  - **装备/药水回收**：40% 售价回收（诅咒装备花费 25 金币处理）
  - **净化/修理**：消除永久 debuff
  - **手动存档**：花费金币保存进度
- 不出售内容：
  - 不卖卡牌（卡牌池固定不变）
  - 不提供删牌服务

## 局外成长（Meta Progression）

游戏失败后重新开始，但以下进度会保留：

| 成长项 | 说明 |
|--------|------|
| 随从解锁 | 通关对应章节后永久解锁新的随从可供选择 |
| 大招自动解锁 | 局外解锁角色后，对应大招开局即可用 |
| 初始属性增加 | 累计游玩可逐步提升随从的初始属性 |
| 初始 ER 提升 | 累计游玩可提升初始每回合能量恢复 |

> 更多局外成长要素待设计，如：
> - 解锁新的卡牌池
> - 成就系统
> - 挑战模式
> - 具体的货币、升级路径和解锁条件

## 单局流程

```
序章（线性教程，局外解锁灼夜后可跳过）：
  玲纱单人教程战 → 遇到灼夜 → 双人教程战
     ↓
第 1-5 章（每章 13 步，每步二/三选一推进）：
  节点完成 → (概率随机事件) → (可使用药水) → 选下一节点(模糊描述+buff/debuff)
  → 进入节点(获得描述中的buff/debuff) → 战斗/商店/事件
  → 循环直到 Boss → 击败 Boss → 自动存档 → 进入下一章
     ↓
  第 5 章 Boss → 通关
     ↓
  任何时候全灭 → 失败 → 局外成长结算 → 返回主界面
```
