# 药水系统设计

> 版本：v1.1
> 状态：框架、药水列表和战斗消耗药水已确定
> 更新日期：2026-05-15

## 概述

药水分为两类：

- **永久成长药水**：在节点之间使用，提供永久且不可撤销的成长效果。
- **一次性战斗药水**：在战斗中、我方 `player_action` 阶段使用，立即消耗，提供即时治疗或本场战斗增益。

永久成长药水与装备（可替换）和卡牌升级（改变现有牌）不同，是对卡组和角色的永久改造。一次性战斗药水则是战斗内资源，不写回 run 成长。

## 基础规则

- **背包上限 3 瓶**（全队共享，非每人 3 瓶）。
- **药水使用后从背包移除**。永久成长药水和一次性战斗药水都属于消耗品。
- **使用时机由 `PotionDef.useTiming` 决定**：
  - `between_nodes`：只能在节点间药水窗口使用；该窗口在核心流程中的位置见 [Run 流程规格](../06-implementation-spec/run-flow.md#核心玩法循环)。
  - `player_action`：战斗中我方行动阶段使用，不消耗 AS，不消耗或产生能量。
- **永久成长药水效果永久不可撤销**：属性成长、新增卡牌、改变卡牌效果均为永久。
- **一次性战斗药水效果只在当前战斗内生效**，治疗为即时效果，属性增益持续到本场战斗结束。
- **背包满时获得新药水**：选择丢弃一瓶旧的，或放弃新药水
- **药水需要选择目标角色**：使用时选择给哪个角色生效

## 稀有度分层

永久成长药水的品质定位如下：

| 品质 | 效果类型 | 卡牌影响 |
|------|---------|---------|
| **普通** | 固定属性成长 | 无 |
| **稀有** | 百分比属性成长 | 可能添加普通强度新卡 或 小幅改变卡牌效果 |
| **史诗** | 较强属性成长 | 可能添加强力新卡 或 显著改变卡牌效果 |
| **传说** | 极强永久效果 | 添加极强新卡 / 大幅改变卡牌 / 复制卡牌 / 删除卡牌 / AS+1 |

一次性战斗药水目前只开放普通和稀有两档。

### 永久药水添加的卡牌 vs 装备添加的卡牌

| | 药水添加 | 装备添加 |
|--|---------|---------|
| 持久性 | 永久，不可撤销 | 跟随装备，替换则移除 |
| 设计定位 | "长"出来的牌 | "借"来的牌 |
| 决策重量 | 极重（用了就是一辈子） | 中等（以后可以换装备换掉） |

## 获取途径

- **商店购买**：商店药水商品规则见 [商店系统](shop.md)
- **战斗掉落**：普通战和精英战概率掉落
- **节点结束后随机事件**：偶尔给药水

## 使用时机

药水只定义自己的可用窗口，不重复定义完整 run 流程：

- `useTiming = "between_nodes"`：在 [Run 流程规格](../06-implementation-spec/run-flow.md#节点间药水窗口) 定义的节点间药水窗口使用。
- `useTiming = "player_action"`：在战斗 `player_action` 阶段使用；具体动作流程见 [战斗状态机](../06-implementation-spec/battle-state-machine.md#使用战斗药水)。

战斗药水使用规则：

1. 只能在 `player_action` 阶段使用。
2. 不消耗目标角色的 AS，不要求目标角色剩余 AS 大于 0。
3. 不消耗公共能量，也不会产生能量。
4. 使用后立即从药水背包移除。
5. 需要目标时，只能选择存活的我方队伍成员。
