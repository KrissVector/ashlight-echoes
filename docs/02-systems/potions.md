# 药水系统设计

> 版本：v1.0
> 状态：框架和药水列表已确定
> 更新日期：2026-03-05

## 概述

药水提供**永久且不可撤销**的成长效果。与装备（可替换）和卡牌升级（改变现有牌）不同，药水是对卡组和角色的永久改造。

## 基础规则

- **背包上限 3 瓶**（全队共享，非每人 3 瓶）
- **只能在节点之间使用**（非战斗中）：节点完成 → 随机事件 → **可使用药水** → 选下一节点
- **效果永久不可撤销**：属性成长、新增卡牌、改变卡牌效果均为永久
- **背包满时获得新药水**：选择丢弃一瓶旧的，或放弃新药水
- **药水需要选择目标角色**：使用时选择给哪个角色生效

## 稀有度分层

| 品质 | 效果类型 | 卡牌影响 |
|------|---------|---------|
| **普通** | 固定属性成长 | 无 |
| **稀有** | 百分比属性成长 | 可能添加普通强度新卡 或 小幅改变卡牌效果 |
| **史诗** | 较强属性成长 | 可能添加强力新卡 或 显著改变卡牌效果 |
| **传说** | 极强永久效果 | 添加极强新卡 / 大幅改变卡牌 / 复制卡牌 / 删除卡牌 / AS+1 |

### 药水添加的卡牌 vs 装备添加的卡牌

| | 药水添加 | 装备添加 |
|--|---------|---------|
| 持久性 | 永久，不可撤销 | 跟随装备，替换则移除 |
| 设计定位 | "长"出来的牌 | "借"来的牌 |
| 决策重量 | 极重（用了就是一辈子） | 中等（以后可以换装备换掉） |

---

## 获取途径

- **商店购买**：每个商店随机刷新 1-3 瓶药水
- **战斗掉落**：普通战和精英战概率掉落
- **节点事件**：战后随机事件偶尔给药水

## 使用时机

```
节点完成 → 概率触发随机事件 → 【可使用药水】 → 选择下一节点 → 选择阵容 → 进入战斗
```

## 传说药水（5种）

极强效果，一局 run 中极其稀有。

| # | 名称 | 效果 |
|---|------|------|
| 1 | 卡牌复制药水 | 选择该角色一张卡牌，复制一份永久加入牌池 |
| 2 | 卡牌消除药水 | 选择该角色一张卡牌，从牌池中永久移除 |
| 3 | 万能改造药水 | 选择该角色一张卡牌，永久费用-1（最低 0） |
| 4 | 天赋觉醒 | 该角色所有属性+10%（ATK/AP/HP/DEF 各+10%） |
| 5 | 极限突破 | 该角色 AS 上限永久+1。添加「超越极限」：0费消耗型，本回合 AS 上限再+1 |
