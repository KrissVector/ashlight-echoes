# Documentation

Start with [00-overview.md](00-overview.md). The documents are organized by purpose:

- `01-world/`: world setting and narrative baseline.
- `02-systems/`: playable systems and rules.
- `03-content/`: characters, enemies, and build archetypes.
- `04-art/`: art direction and concept references.
- `05-data-drafts/`: migrated JSON drafts for reference.

The old prototype code and old technical architecture were not migrated.
