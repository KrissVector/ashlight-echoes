## 一、整体风格

- **风格定位**：高品质二次元（Premium Anime Style），参考《鸣潮》《崩铁》角色设计品质
- **画面基调**：精致少女、细节丰富、光影精致、服装设计感强
- **年龄感**：16-20 岁精致少女，兼顾优雅与青春感（珂莱塔式少女感，非过度成熟）
- **核心关键词**：精致不幼态、优雅不老成、时尚+幻想融合、高视觉密度
- **绝对禁止**：Q 版/萌系/低头身比/过大眼睛/儿童体型/简化卡通画风/过度性感
- **概念图生成方式**：方案 B — 参考图+文字描述多模态输入（Gemini 2.5 Flash）

## 三、角色设计规范

### 3.1 基本参数

| 项目 | 规格 |
|------|------|
| 头身比 | **8-9 头身**（全身立绘），战斗中使用**半身像**（非 Q 版） |
| 身高范围 | 160-175cm（所有角色均为成年女性体型） |
| 面部比例 | 成熟二次元比例——眼睛精致但不过大，有明确下颌线和鼻梁 |
| 线条风格 | 精细线条，注重面料质感和金属光泽的表现 |
| 阴影风格 | 渐变软阴影为主+关键结构用硬切阴影，注重光影层次 |
| 眼睛 | 精致有神，瞳孔有多层色彩和高光，体现角色气质而非"萌感" |
| 身材 | 成熟女性体型，体态优雅，比例协调（参考鸣潮角色身材） |

### 3.2 设计原则

- **所有可玩角色均为女性**
- 角色设计追求**高时尚感**——幻想元素与现代服装设计感融合
- 服装设计需有**高视觉密度**——多层次、多材质、精致配饰
- 每个角色有独特的**气质定位**（冷艳/热情/神秘/温柔/冷酷）
- 武器/装备精致且有存在感，不能是简单的道具
- 角色剪影必须可辨识——仅看黑色轮廓即可区分每个角色
- 发型设计是辨识度核心——每个角色发型独特且有记忆点
- 配色使用专属色系，避免角色间色彩混淆

### 3.3 鸣潮风格关键要素

| 要素 | 说明 |
|------|------|
| 服装层次 | 外套/披风 + 内衬 + 配饰 + 绑带/腰封，至少 3 层视觉层次 |
| 材质表现 | 需体现不同材质——丝绸、金属、皮革、轻纱的质感差异 |
| 光效点缀 | 角色身上有与属性相关的微弱光效或能量痕迹 |
| 配饰细节 | 耳饰、发饰、腰饰、臂环等精致小配件增加细节密度 |
| 色彩饱和度 | 中高饱和度，不过度鲜艳也不灰暗，追求"高级感" |

### 3.4 敌人设计规范

| 敌人类型 | 体型 | 设计要点 |
|---------|------|---------|
| 普通小怪 | 较小/正常 | 简单剪影、低细节、批量可复用 |
| 精英怪 | 正常/偏大 | 有标志性装备或特征、中等细节 |
| Boss | 大型/巨型 | 高细节、威压感、多个视觉焦点、阶段变化 |

## 四、卡牌美术规范

### 4.1 卡牌尺寸与构图

| 项目 | 规格 |
|------|------|
| 卡牌比例 | 2:3（竖版） |
| 卡牌插画区域 | 占卡面上方 60% |
| 文字信息区域 | 占卡面下方 40%（名称、费用、效果描述） |
| 费用图标位置 | 左上角圆形区域 |
| 稀有度标识位置 | 卡牌底部边框 |

### 4.2 稀有度视觉区分

| 稀有度 | 边框样式 | 边框色 | 背景效果 |
|--------|---------|--------|---------|
| 普通 | 简洁直线边框 | 灰色 #808080 | 无特效 |
| 稀有 | 内侧发光边框 | 蓝色 #3498DB | 微弱光晕 |
| 史诗 | 双层装饰边框 | 紫色 #9B59B6 | 粒子飘散 |
| 传说 | 金属质感浮雕边框 | 金色 #FFD700 | 星辉流光 |

### 4.3 卡牌插画风格

- 插画以角色动作/技能施放瞬间为主题
- 角色在卡牌中保持**成熟画风**（与立绘风格一致）
- 背景使用虚化的技能特效或场景
- 物理卡牌偏暖色调（红/橙/金），法术卡牌偏冷色调（蓝/紫/白）
- 辅助卡牌使用绿/金色调
- 消耗性卡牌插画增加"燃烧/碎裂"的视觉暗示

## 五、战斗 UI 风格

> 参考素材：`docs/04-art/battle-ui-reference.png`

### 5.1 设计理念

- **"卡牌即角色"**：敌人和我方角色统一用卡牌形态展示，视觉语言统一
- **舞台式纵向分层**：敌人卡(上) → 角色卡(中) → 手牌(下)
- **暗底亮框**：深色竞技场背景让卡牌"浮"在上面，圆角边框+描边突出信息
- 左下角玩家面板（头像+等级+能量+金币），不占战场空间

### 5.2 布局结构

- 横屏布局（1920×1080 基准，Canvas 自适应）
- **上方**：敌人卡牌区（虚空紫边框）
- **中间**：角色卡牌区（星辉金边框），每张卡含占位插画区+名称+HP+AS
- **下方**：手牌区（圆角卡牌框，左上角费用标记）
- **左下角**：玩家面板（头像占位+Lv+⚡能量+💰金币）
- **右侧**：结束回合按钮
- **右上角**：回合数+战斗日志
- **中央竞技场**：深色圆角矩形背景（#14121E），营造暗色舞台氛围

### 5.3 卡牌面板规格

| 类型 | 尺寸 | 边框色 | 填充色 | 圆角 |
|------|------|--------|--------|------|
| 角色卡 | 105×140 | 星辉金 #FFD700 | 深蓝 rgba(25,35,55,0.9) | 6px |
| 敌人卡 | 100×130 | 虚空紫 #6B2FA0 | 暗紫 rgba(80,30,60,0.9) | 6px |
| 手牌 | 110×85 | 灰紫 rgba(80,70,100) / 选中金 | 深色 rgba(30,28,40,0.96) | 5px |
| 玩家面板 | 110×120 | 星辉金 | 极深 rgba(15,12,25,0.9) | 8px |

### 5.4 UI 色调

- 整体基调：极深色背景 #0C0A12 + 深色竞技场 #14121E
- 文字：白色/浅灰为主，关键信息用星辉金高亮
- 按钮：结束回合=焰境红边框，普攻=翠梦绿
- HP 用 ♥ 前缀，护盾用 🛡 前缀，能量用 ⚡ 前缀

## 七、概念图提示词风格指引

为保证 AI 生成的概念图风格统一，所有提示词应遵循以下结构：

```
[Style prefix] + [Character/Object description] + [Action/Pose] + [Background] + [Quality suffix]
```

### 风格前缀（统一使用——鸣潮风格）

```
wuthering waves style, premium anime character illustration,
mature sophisticated character design, high-end gacha game character art,
detailed costume design with multiple fabric layers and accessories,
refined facial features with elegant proportions
```

### 品质后缀（统一使用）

```
cinematic lighting, sharp focus, rich color palette,
intricate details, professional illustration quality,
8K render quality concept art
```

### 敌人特殊修饰

```
dark fantasy, ominous atmosphere, void corruption visual,
purple-black energy veins, menacing presence,
dramatic lighting contrast
```
