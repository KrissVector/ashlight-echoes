# Ashlight Echoes

A 2D anime-style roguelike deckbuilder set in the post-Starfall world of Hui Jin / Ashlight, where a rotating cast of characters searches ruined regions for Lumina Shards and fights the spreading Void.

This repository is the formal project home. The current migration intentionally brings over design documents, concept art, and configuration drafts only. Legacy prototype code remains in the old project and is not part of this codebase.
